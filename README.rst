===============================================================================
Miroslava
===============================================================================

If `Flask <https://flask.palletsprojects.com/en/stable/>`_ is a lightweight web
application framework, Miroslava is even more lightweight.

-------------------------------------------------------------------------------
Installation
-------------------------------------------------------------------------------

.. See more at: https://stackoverflow.com/a/15268990

Install the latest version of Miroslava using `pip`_:

.. code-block:: bash

    pip install -U git+https://github.com/xames3/miroslava.git
